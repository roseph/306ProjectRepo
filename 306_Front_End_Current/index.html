About UREC Count IT

UREC Count IT is an application that provides users of the JMU UREC facilities an accurate indication of how many individuals are in a given room at one time. This allows them to make decisions on whether to go and utilize the facility, or wait and find a time that may be less crowded. The application offers a live map of the building that indicates the current number of users in a given room. It also features historical data for finding the best times to go, and a reservation system that allows for users to reserve individual rooms.
